# 云栈 - 项目文档（完整版）

## 项目简介

云栈（yunzhan）是一个**完全免费、离线可用的云计算学习桌面应用**。基于 Electron + Vue 3 构建，**20 门课程**覆盖 Linux、Docker、Kubernetes、数据库、网络、安全、DevOps 等运维全栈技能，含一门**企业级实战项目**。

## 核心功能

### 1. 课程学习
- 20 门课程，从入门到高级完整路径
- 每门课程 5-14 个章节
- 内容从 .ts 文件提取为独立 .md 文件，按需加载（首屏 JS 从 1MB+ → 200KB）
- **手动确认进度** — 点击「标记已学完」弹出确认框，可取消
- 侧边栏显示章节列表和完成状态（绿色勾）
- **骨架屏** — 课程加载时显示内容占位动画

### 2. 知识点讲解库（202条）
- ⭐ **必修标记** — 核心知识点带金色徽章标识
- 覆盖 Linux 命令、Docker、K8s、网络、存储、进程、权限等
- 全景覆盖 Word 教程（云计算逆战课堂第01-06章）全量知识点
- 弹窗显示分类、难度、关联概念、代码示例

### 3. 代码命令讲解
- Markdown 代码块增强：hover 和点击弹出命令详解
- `explainer.ts` 识别代码中的系统命令（40+ 命令）
- `knowledge.ts` 知识库（202条）支持关键词匹配

### 4. 全局搜索（Ctrl+K）
- 跨所有课程全文搜索
- 键盘上下选择，回车跳转

### 5. 学习统计
- 连续学习天数（自动计算）
- 总学习天数、今日是否已学习
- 课程完成度、答题准确率、成就系统
- **导出学习进度 JSON**

### 6. 书签/收藏
- 每章节可收藏，首页和进度页显示

### 7. 主题切换
- 日间/夜间模式（CSS 变量驱动，持久化到 localStorage）
- Tailwind 语义化颜色（surface-*, ink-*, edge-*）

### 8. 实战项目：企业级 DevOps 平台部署
- Docker Compose 7 容器编排
- Nginx 反向代理 + WordPress + MySQL + Redis
- Prometheus + Grafana 全栈监控（4 条告警规则）
- Shell 自动备份/健康检查/一键部署脚本
- **可直接写入简历的项目描述**

## 数据流

```
课程数据 → courses/*.ts（元数据 + contentFile引用）
         → markdown/*.md（懒加载，?raw 静态导入）
         → contentLoader.ts（contentIndex 索引表）
         → CourseDetailPage（loadContent() 异步加载）

学习进度 → Pinia Store → localStorage（'ops-school-progress'）
         自动记录 trackStudyDay()，手动点击标记章节完成

知识讲解 → knowledge.ts（202条，core:true 必修标记）
         → MarkdownRenderer（标注 <span class="knowledge-term">）
         → ConceptPopover（悬停弹窗，⭐ 必修徽章）
```

## 数据类型

```typescript
interface LearningProgress {
  completedChapters: Record<string, number[]>  // { courseId: [0, 1, 3] }
  quizRecords:       Record<string, QuizRecord>
  achievements:      string[]
  lastVisited:       string                     // 最近访问课程 ID
  totalTimeSpent:    number
  bookmarks:         Bookmark[]
  studyDays:         string[]                   // ISO 日期字符串
}

interface KnowledgeEntry {
  term:        string          // 术语名称
  keywords:    string[]        // 匹配关键词
  summary:     string          // 一句话概述
  description: string          // 详细讲解
  category:    string          // 所属领域
  level?:      'beginner' | 'intermediate' | 'advanced'
  core?:       boolean         // ⭐ 必修标记
  related?:    string[]        // 关联概念
  tips?:       string          // 实操提示
  refUrl?:     string          // 参考链接
  example?:    string          // 代码示例
}
```

## 构建与部署

```bash
npm run dev            # 开发服务器
npm run build:fast     # 跳过类型检查，快速打包
npm run build          # 含类型检查的完整构建
npm run electron:build # 打包 Electron 桌面应用

# 提取课程 Markdown 内容
node scripts/extract-markdown.mjs
```

输出：`release/win-unpacked/` — 绿色免安装文件夹。
