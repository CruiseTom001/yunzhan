## 云上安全最佳实践

### 责任共担模型

云厂商负责"云的安全"，客户负责"云中的安全"。

### RAM 权限管理

- 最小权限原则（Least Privilege）
- 使用 RAM 角色而非 AccessKey
- 定期轮换 AccessKey（90 天）

### 安全组最佳实践

- 最小开放（白名单模式）
- 分层隔离
- 使用安全组引用而非 IP

### WAF（Web 应用防火墙）

- SQL 注入防护
- XSS 跨站脚本防护
- CC 攻击防护

### 数据加密

- 传输加密（TLS/mTLS）
- 存储加密（SSE-KMS, TDE）
- 密钥管理（KMS）
