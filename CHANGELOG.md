# 变更日志

本文件遵循 [Keep a Changelog](https://keepachangelog.com/) 简化版,每条变更在括号内标注功能象限:
A = 必须同步(进度/实验/IPC/Schema) · B = 推荐同步(UI/内容) · C = 桌面专属 · D = 桌面跳过。
