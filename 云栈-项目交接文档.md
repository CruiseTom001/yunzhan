## ⚠️ 给新 Agent 的注意事项

1. **knowledge.ts 非常脆弱**——被 Python 脚本重建过多次，不要再用 Python 批量修改它，手动编辑更安全
2. **FloatingTerminal.vue 容易出语法错误**——改终端代码时先确保括号配对（40 个 { 和 40 个 }）
3. **新装依赖**：npm 已装 `monaco-editor` `@xterm/xterm` `@xterm/addon-fit`，不要重复安装
4. **contentLoader.ts 使用静态 `import 'xxx.md?raw'`** 模式加载 Markdown，新增课程需同步更新导入语句
5. **端口可能被占满**——先 `taskkill /f /im node.exe` 释放一下再启动
6. **课程内容在 src/data/markdown/** 下，课程元数据在 src/data/courses/*.ts
