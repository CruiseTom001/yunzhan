## LVL.1 夯实基础（入门 · 46 学时）

### 1. 计算机基础与操作系统原理（12h）

| 章节 | 核心内容 |
|------|---------|
| 计算机体系结构 | CPU 架构、内存层次、磁盘 SSD/HDD、网络硬件、性能瓶颈分析 |
| 操作系统概述 | 内核/用户态、Ring 0-3、系统调用、中断与异常、Linux 发行版 |
| 进程与线程 | 进程状态、fork+exec、CFS 调度、信号机制、僵尸进程、IPC |
| 内存管理 | 虚拟内存、分页机制、多级页表、Swap、mmap、OOM Killer |
| 文件系统原理 | ext4/xfs、inode、超级块、挂载 fstab、软硬链接原理 |
| 系统启动流程 | BIOS/UEFI、MBR/GPT、GRUB2、initramfs、systemd target |

### 2. Linux 系统基础（20h）

| 章节 | 核心内容 |
|------|---------|
| 文件与目录操作 | ls/cd/pwd/mkdir/rm/cp/mv/find/locate/tree/ln/stat |
| 文件查看与编辑 | cat/less/head/tail/grep/sed/awk/wc/sort/uniq/vim |
| 权限管理 | chmod/chown/chgrp/sudo/useradd/userdel/groupadd/ACL/umask |
| 进程管理 | ps/top/htop/kill/pkill/nice/renice/systemctl/nohup/jobs/bg/fg |
| 网络相关 | ping/curl/wget/netstat/ss/ifconfig/ip/dig/nslookup/tcpdump/iptables |
| 磁盘与存储 | df/du/mount/umount/fdisk/parted/tar/rsync/scp/lvm/ln |
| 包管理 | apt/yum/dnf/rpm/dpkg/repo 配置/源码编译 |
| 系统信息与监控 | uname/free/vmstat/iostat/mpstat/dmesg/lscpu/lsmem |
| Shell 脚本编程 | 变量、条件判断、循环、函数、数组、正则表达式、调试技巧 |
| 定时任务 | crontab/at/batch/systemd timer/logrotate |

### 3. 网络基础（15h）

| 章节 | 核心内容 |
|------|---------|
| 网络模型 | OSI 七层/TCP/IP 四层、数据封装解封装、各层功能与协议 |
| IP 与子网 | IPv4/IPv6、子网掩码/CIDR、网关/路由、NAT 原理 |
| TCP/UDP | 三次握手/四次挥手、滑动窗口、拥塞控制、UDP 适用场景 |
| DNS | 域名解析流程、A/CNAME/MX/TXT 记录、dig/nslookup、DNS 缓存 |
| HTTP/HTTPS | 请求方法/状态码、Cookie/Session、HTTPS/TLS 握手、HTTP/2/3 |
| 网络排查 | ping/traceroute/mtr、tcpdump/Wireshark、netstat/ss、curl -v |

### 4. Git 版本控制（8h）

| 章节 | 核心内容 |
|------|---------|
| Git 基础概念 | 版本控制演进、三区域模型、Blob-Tree-Commit 数据模型、.gitignore |
| Git 常用命令 | init/clone/add/commit/status/diff/log/push/pull/fetch/merge |
| 分支管理 | switch/restore、Git Flow vs GitHub Flow、rebase vs merge、冲突解决 |
| 远程协作 | remote/upstream、远程分支、PR 工作流、Code Review、团队规范 |
| Git 进阶 | stash/cherry-pick/reset/revert/bisect/tag 语义化版本/reflog |
| Git 在运维中的应用 | etckeeper、IaC (Terraform/Ansible)、CI 集成、Git Hooks、敏感信息管理 |

| 日志与配置管理 | logging 日志轮转、configparser/YAML/JSON、环境变量覆盖 |
| 常用运维库 | psutil 系统监控、APScheduler 定时、邮件告警 |
| Python 脚本实战 | 批量部署脚本、日志分析脚本、服务器巡检脚本 |

| 虚拟化在运维中的应用 | Vagrant 开发环境、测试环境、资源隔离、Overcommit |
