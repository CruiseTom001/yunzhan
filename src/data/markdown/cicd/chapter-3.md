## Jenkins Pipeline

Jenkins 是最流行的开源 CI/CD 工具之一。Pipeline 使用 Groovy DSL 定义构建、测试、部署流程。

### Jenkinsfile 基础

使用 Declarative Pipeline 语法定义流水线。

### 多阶段部署 Pipeline

支持并行测试、Kubernetes Agent、审批门禁等高级功能。

### Scripted Pipeline 示例

使用脚本式 Pipeline 实现更灵活的流水线控制。

### 共享库（Shared Library）

将常用流水线逻辑封装为可复用的共享库。

### Jenkins Docker 部署

使用 Docker Compose 部署 Jenkins 和相关组件。
