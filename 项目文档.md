# 云栈 - 项目文档

## 项目简介

云栈（yunzhan）是一个**完全免费、离线可用的云计算学习桌面应用**。基于 Electron + Vue 3 构建，19 门课程覆盖 Linux、Docker、Kubernetes、数据库、网络、安全等运维全栈技能。

## 技术栈

| 技术 | 用途 |
|------|------|
| **Vue 3** (Composition API + `<script setup>`) | UI 框架 |
| **Vite 5** | 构建工具 |
| **TypeScript** (~5.3) | 类型安全 |
| **Pinia** | 状态管理（持久化到 localStorage） |
| **Vue Router** | 路由（Hash 模式） |
| **Tailwind CSS 3** | 样式（任意值 `bg-[#...]` 大量使用） |
| **Electron 35** | 桌面壳（绿色免安装，仅打包前端 dist） |
| **lucide-vue-next** | 图标库 |
| **markdown-it** | Markdown 渲染 |
| **highlight.js** | 代码高亮 |
| **vue-tsc** | 类型检查（v1 版本，可能不兼容 TS 5.3） |

## 页面路由

| 路径 | 页面 | 说明 |
|------|------|------|
| `/` | 首页 | 英雄区 + 继续学习 + 技能树 + 核心特性 |
| `/courses` | 课程列表 | 搜索/筛选/分类展示 19 门课程 |
| `/course/:id/chapter/:idx` | 课程详情 | 侧边栏 + Markdown 内容 + 进度/收藏 |
| `/quiz` | 答题练习 | 分类刷题 |
| `/progress` | 学习进度 | 数据统计 + 课程进度 + 成就系统 + 学习天数 |

## 核心功能

### 1. 课程学习
- 19 门课程，从入门到高级的完整路径（4 个阶段：基础→核心→进阶→架构）
- 每门课程 5-14 个章节，内容为 Markdown 格式
- **手动确认进度** — 读完需点击「标记已学完」，弹出确认框，可取消
- 侧边栏显示章节列表和完成状态（绿色勾）

### 2. 代码命令讲解
- Markdown 代码块增强：hover 和点击弹出命令详解
- `explainer.ts` 识别代码中的系统命令，匹配 `knowledge.ts` 知识库
- 知识库含 40+ 知识点：Linux 命令、Docker、K8s、Git、网络、数据库等

### 3. 全局搜索 (Ctrl+K)
- 跨所有课程全文搜索
- 先搜元数据（标题/描述），再懒加载搜索章节内容
- 键盘上下选择，回车跳转
- 支持搜索建议标签

### 4. 学习统计
- 连续学习天数（自动计算）
- 总学习天数
- 今日是否已学习（自动记录）
- 课程完成度、答题准确率、成就系统

### 5. 书签/收藏
- 每章节可收藏，首页和进度页显示收藏列表

### 6. 主题切换
- 日间/夜间模式切换（按钮在右上角）
- CSS 变量驱动，持久化到 localStorage

### 7. 导出进度
- 进度页右上角「📥 导出进度」，下载 JSON 格式学习报告

## 数据流

```
课程数据 → courses/*.ts（独立文件，懒加载）
       ↓
all.ts（懒加载注册表 → import() 动态导入）
       ↓
CourseDetailPage（onMounted → getCourse(id)）

学习进度 → Pinia Store → localStorage（'ops-school-progress'）
       ↓
每个课程访问自动记录 trackStudyDay()
手动点击「标记已学完」记录章节完成
```

## 数据类型 (LearningProgress)

```typescript
interface LearningProgress {
  completedChapters: Record<string, number[]>  // { courseId: [0, 1, 3] }

```

## 构建与部署

```bash
npm run dev            # 开发服务器 (localhost:5173)
npm run build:fast     # 跳过类型检查，快速打包
npm run build          # 含类型检查的完整构建
npm run electron:build # 打包 Electron 桌面应用
```

输出：`release/win-unpacked/` — 绿色免安装文件夹，压缩后发给用户直接使用。

## 已知限制 / 潜在优化方向

1. **vue-tsc 版本** — 降级到 v1（^1.8.27）以兼容当前 TypeScript 5.3，可升级 TS 和 vue-tsc
2. **Tailwind 任意值** — 大量使用 `bg-[#...]` 和 `border-white/[0.03]` 等，主题切换靠 CSS `!important` 覆盖，不够优雅。可考虑系统化改用 CSS 变量
3. **课程数据量** — 每个课程 .ts 文件包含完整 Markdown 内容（linux-basics.ts 1699 行），数据与展示未分离。可迁移为 JSON/Markdown 文件
4. **Electron 打包** — 当前仅打包为 `win-unpacked` 目录（因权限问题无法生成安装包），可配置代码签名或管理权限
5. **无后端** — 所有数据在本地，无云同步。学习进度只存在 localStorage，清除浏览器数据会丢失
6. **测试覆盖** — 无单元测试/E2E 测试
7. **响应式** — 桌面端为主，移动端未充分适配
8. **无障碍** — 未考虑 a11y（aria 标签、键盘焦点管理）
9. **i18n** — 界面中英文混合（按钮英文 `BACK`、`NEXT` 与中文混用）
10. **代码拆分** — CourseDetailPage chunk 仍有优化空间（highlight.js、markdown-it 可拆为 vendor）
